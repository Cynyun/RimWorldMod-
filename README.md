# RimWorldModManager

使用 SteamCMD 下载和更新 RimWorld Mod 的管理器。

## 功能特性

- 双视图管理：本地游戏 Mod 和 Workshop Mod
- 通过 Workshop ID 下载 Mod
- 批量更新选中的 Workshop Mod
- Mod 详情预览：名称、作者、版本、描述及预览图片
- 搜索过滤：快速搜索和过滤 Mod 列表
- 打开文件夹：直接打开 Mod 所在文件夹
- 打开 Workshop：跳转到 Mod 的 Steam Workshop 页面
- 智能冲突处理：导入时自动检测同名 Mod
- 批量下载：支持输入多个 Workshop ID 批量下载

## 系统要求

- Windows 10/11
- .NET 10.0 Runtime
- SteamCMD

## 使用说明

1. 下载 SteamCMD 并解压到程序目录的 `steamcmd` 文件夹
2. 运行 `RimWorldModManager.exe`
3. 在设置中配置 SteamCMD 路径和游戏 Mod 路径
4. 使用搜索框过滤 Mod，点击下载/更新按钮进行操作
